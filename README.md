# 📚 Bir Kitap Bir Gelecek

Köy okulları için geliştirilen masaüstü kütüphane otomasyon sistemidir. 
Öğrencilerin kitapları görüntüleyebilmesi, ödünç alabilmesi ve iade edebilmesi sağlanırken; yönetici paneli üzerinden kitap, öğrenci ve ödünç işlemleri yönetilebilmektedir.

## ✨ Özellikler

- 👨‍🎓 Öğrenci girişi
- 🔐 Yönetici girişi
- 📖 Kitap listeleme
- ➕ Kitap ekleme
- ✏️ Kitap güncelleme
- ❌ Kitap silme
- 📚 Ödünç alma işlemleri
- 🔄 Kitap iade işlemleri
- 🔍 Kitap arama
- 🗄️ SQLite veritabanı desteği

## 🛠️ Kullanılan Teknolojiler

- Java
- Java Swing
- SQLite
- JDBC
- OOP (Nesne Yönelimli Programlama)
